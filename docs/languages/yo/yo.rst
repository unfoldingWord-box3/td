Yorùbá
========

.. note:: The `Ethnologue <https://www.ethnologue.com/language/yor>`_ identifies this language as ``yor``.

This language is spoken in the following countries:

* BJ: Benin
* BF: Burkina Faso
* GQ: Equatorial Guinea
* GH: Ghana
* NE: Niger
* NG: Nigeria
* US: United States of America
* TG: Togo
* SL: Sierra Leone
